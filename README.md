# Registration

A simple javascript form with following attributes-
 1. First Name, Last Name </br>
 2. Email id </br>
 3. Phone no. </br>
 4. Date of birth </br>
 5. Password </br>
 6. Register button</br>

No database, just created a javascript form with UI.
